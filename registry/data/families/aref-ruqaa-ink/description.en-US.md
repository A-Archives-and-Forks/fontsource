Aref Ruqaa Ink is an Arabic typeface that aspires to capture the essence of the classical Ruqaa calligraphic style.

This font uses the COLRv1, CPAL and SVG tables. Please visit [the gf-guide color page](https://googlefonts.github.io/gf-guide/color.html) to learn more about Color Fonts technology.

The Aref Ruqaa project is led by Khaled Hosny, a type designer based in Cairo, Egypt. To contribute, see [github.com/alif-type/aref-ruqaa](https://github.com/alif-type/aref-ruqaa)
