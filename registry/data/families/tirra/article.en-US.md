Tirra is a Unicode font family for rendering characters in languages using the Neo-Tifinagh alphabet. This font is designed to reflect a handwriting style. It is a subset of [the Akatab](https://fonts.google.com/specimen/Akatab) font family specifically for languages using Neo-Tifinagh, and has the same six weights: Regular, Medium, SemiBold, Bold, ExtraBold, and Black.

To contribute, see [github.com/silnrsi/font-akatab](https://github.com/silnrsi/font-akatab).
