Slabo is a collection of size-specific fonts for use in online advertising and other web uses. The collection currently includes this font, Slabo 13px, and [Slabo 27px](http://www.google.com/fonts/specimen/Slabo+27px). Each font in the collection is fine-tuned for use at the pixel size in its name.

The Slabo project is led by Tiro Typeworks, a type design foundry based in Canada. To contribute, see [Slabo](https://github.com/TiroTypeworks/Slabo) on GitHub.
