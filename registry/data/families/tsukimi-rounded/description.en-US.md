Tsukimi Rounded is kana font that is based san-go(Japanese traditional go-number sized type, #3) in Tsukiji Type Foundry’s specimen books. Original san-go type is Mincho style but Tsukimi Rounded is sans-serif typeface with rounded terminals.

To contribute to the project, visit [github.com/mt-funa/Tsukimi-Rounded](https://github.com/mt-funa/Tsukimi-Rounded)
