Abyssinica SIL is a Unicode font that supports Ethiopic and Latin languages.

For more information please visit the Abyssinica SIL page on SIL International's Computers and Writing systems website, [scripts.sil.org/AbyssinicaSIL](http://scripts.sil.org/AbyssinicaSIL)

To contribute, see [github.com/silnrsi/font-abyssinica](https://github.com/silnrsi/font-abyssinica)
