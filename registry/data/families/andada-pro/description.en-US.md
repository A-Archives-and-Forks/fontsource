Andada Pro is an organic-slab serif, hybrid style and medium contrast type for text, initially designed to be used in a specific bilingual context, Spanish and Guaraní (pre-hispanic) languages. This font has received an award at the Ibero-America Design Biennial. The Biennial has been shown in Spain, Argentina, Chile, El Salvador, Uruguay, Bolivia, Colombia and Venezuela.

Designed by Carolina Giovagnoli for [HT Fonts](https://www.htfonts.com/).

To contribute see [github.com/huertatipografica/Andada-Pro](https://github.com/huertatipografica/Andada-Pro).
