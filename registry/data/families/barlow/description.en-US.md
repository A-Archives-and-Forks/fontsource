Barlow is a slightly rounded, low-contrast, grotesk type family. Drawing from the visual style of the California public, Barlow shares qualities with the state's car plates, highway signs, busses, and trains.

This is the Normal family, which is part of the superfamily along with [Semi Condensed](http://fonts.google.com/specimen/Barlow+Semi+Condensed) and [Condensed](http://fonts.google.com/specimen/Barlow+Condensed), each with 9 weights in Roman and Italic.

The Barlow project is led by Jeremy Tribby, a designer based in San Francisco, USA. To contribute, see [github.com/jpt/barlow](https://github.com/jpt/barlow)
