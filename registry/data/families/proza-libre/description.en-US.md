Proza Libre is the libre version of the retail Proza type family, by Bureau Roffa. It is made to render exceptionally well on screens across different operating systems, especially Windows, using [ttfautohint](http://freetype.org/ttfautohint).

The Proza Libre project is led by Jasper de Waard, a type designer based in the Netherlands. To contribute, see [github.com/jasperdewaard/Proza-Libre](https://github.com/jasperdewaard/Proza-Libre)
