_Epilogue_ is a sans serif variable font with a weight axis. 9 weights, upright and italic. It supports a wide range of languages in the latin script scope.

The Epilogue project is led by Tyler Finck — type designer running [Etcetera Type Co](https://www.etc.supply/) in Ithaca, New-York, USA. To contribute, see [github.com/Etcetera-Type-Co/Epilogue](https://github.com/Etcetera-Type-Co/Epilogue)
