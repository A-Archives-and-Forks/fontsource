Cairo is a contemporary multilingual typeface family. [Mohamed Gaber](https://gaber.design/) extended the Latin typeface family [Titillum Web](https://fonts.google.com/specimen/Titillium+Web) to support the Arabic script, with a design that is based on the Kufi calligraphic style. Now available as a variable font.

Cairo balances classic and contemporary tastes with wide open counters and short ascenders and descenders that minimize length while maintaining easy readability. The lighter weights can be used for body text while the heavier weights are perfect for headlines and display typography. The Arabic component has a wide glyph set that supports the Arabic, Farsi and Urdu languages.

The Cairo project is led by [Mohamed Gaber](https://gaber.design/), a type designer based in Cairo, Egypt. To contribute, see [github.com/Gue3bara/Cairo](https://github.com/Gue3bara/Cairo)
