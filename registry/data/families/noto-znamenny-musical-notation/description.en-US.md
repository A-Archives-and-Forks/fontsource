Noto is a global font collection for writing in all modern and ancient languages. Noto Znammeny Musical Notation is a font that contains symbols for the Znamenny Chant _musical notation_. It has 527 glyphs.

This font uses the COLRv0 and CPAL tables. Please visit [the gf-guide color page](https://googlefonts.github.io/gf-guide/color.html) to learn more about Color Fonts technology.
