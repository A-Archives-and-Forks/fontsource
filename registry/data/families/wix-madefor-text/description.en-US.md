Wix Madefor is a compact font family of [five weights for display setting,](https://fonts.google.com/specimen/Wix+Madefor+Display/) and four text weights for use in user interfaces. The text fonts also have matching italics, with shapes rooted in cursive forms, despite the typeface's geometric structure, allowing Wix to have different expressions, from functional to more playful.

The typeface was designed to carry subtle features with an engaging and clean appearance.

To contribute, see [github.com/wix/wixmadefor](https://github.com/wix/wixmadefor).
