M+ FONTS is a little nifty font family for everyday usage. Mplus Code Latin is a Sans Serif font with seven weights from Thin to Bold, supporting GF Latin Plus. Because this font is for programming usage, it has high readability even in small sizes, and letterforms are designed to avoid misreadings as much as possible. There are two styles—Mplus Code Latin 50 and Mplus Code Latin 60, each having 50% and 60% character width.

To contribute to the project, visit [github.com/coz-m/MPLUS\_FONTS](https://github.com/coz-m/MPLUS_FONTS)
