[The Generative Fonts](https://www.generativefonts.xyz/) is an original, script-generated collection of fonts based on the Google Fonts [Rubik](https://fonts.google.com/specimen/Rubik) by Hubert and Fischer, Meir Sadan and Cyreal. The code used to generate it can be found at [github.com/NaN-xyz/Glyph-Filters](https://github.com/NaN-xyz/Glyph-Filters).

Check for more [Rubik "Filtered" fonts](https://fonts.google.com/?query=Rubik)!.

To contribute, see [github.com/NaN-xyz/Rubik-Filtered](https://github.com/NaN-xyz/Rubik-Filtered).
