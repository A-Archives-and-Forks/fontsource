Mooli is a Sans Serif font designed for young readers. Mooli is derived from the [Mulish](https://fonts.google.com/specimen/Mulish) font family. It has been designed to be used freely across the internet by web browsers on desktop computers, laptops and mobile devices.

To contribute, please see [github.com/googlefonts/mooliFont](https://github.com/googlefonts/mooliFont).
